## Rana Brothers – Importer, Exporter & Commission Agents

This is a production-ready marketing and lead-generation website for **Rana Brothers**, an agri-supply and commission-based business focused on wholesale fruits and vegetables and export services.

The site is built with **Vite + React**, **React Router**, **Tailwind CSS**, and **react-helmet-async** for basic SEO tags.

### Features

- **Multi-page routing**: Home, About, Products, Fruits, Vegetables, Services, Contact.
- **Products catalog** with search, category pages, and “Get Quote” CTAs that pre-select the product on the Contact form.
- **Home landing page** with hero, credibility highlights, vision section, company video placeholder, company motto cards, CEO section, featured products preview, testimonials, and final CTA banner.
- **Contact page** with validation for required fields, email format checks, and inline success message on submit (no backend).
- **Central content configuration** in `src/data/siteData.js` for navigation, branding, contact info, leadership, products, services, and SEO metadata.
- **UI/UX extras**: sticky navbar with Products dropdown, back-to-top button, floating “Quick Inquiry” WhatsApp button, and mobile-responsive layout.

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Then open the printed local URL (typically `http://localhost:5173/`) in your browser.

### Create a production build

```bash
npm run build
```

The optimized static assets will be generated in the `dist` folder, ready to be deployed to any static hosting provider.
